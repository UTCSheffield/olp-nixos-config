# Update tool Server

## Stack
- Typescript
- Node.js
- Volta
- Express

## Development
### Basic requirements
- [Volta](https://volta.sh/) - Used for pinning exact versions of Yarn, and Node, and any required CLI tools
- A computer
- If doing work on the Nix flakes, [Nix](https://nixos.org) installed
### Installation
1. Run `yarn`
### Building
If building:
1. `yarn build`
2. `yarn start`
